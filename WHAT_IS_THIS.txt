## Install + run

```bash
tar -xzf lakewind-code.tar.gz
cd lakewind_clean
python -m venv .venv && source .venv/bin/activate
pip install -e .

lakewind init-db
lakewind doctor
lakewind backfill --days 60
lakewind retrain --days 60 --backend xgboost_gpu   # for RTX 3070
lakewind promote <model_version> --force
lakewind predict --no-collect
lakewind backtest --days 60 --candidate <model_version>
```
